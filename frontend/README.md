Pizza Store

A full-stack Pizza Ordering Web Application built with React, Vite, Node.js, Express.js, MongoDB, and Firebase Authentication.

🚀 Features
User Registration and Login with Firebase Authentication
Browse a variety of delicious pizzas
Add and remove items from the cart
Increase and decrease pizza quantities
Order management and checkout functionality
Responsive design for desktop and mobile devices
Secure backend API integration
Modern and user-friendly interface
🛠️ Technologies Used
Frontend
React.js
Vite
React Router DOM
Tailwind CSS
Backend
Node.js
Express.js
MongoDB
Authentication
Firebase Authentication
